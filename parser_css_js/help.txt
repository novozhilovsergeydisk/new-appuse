Ваш код на Go выглядит хорошо и выполняет задачу по скачиванию CSS и JS файлов из указанного HTML-файла. Чтобы успешно запустить этот код, вам необходимо убедиться, что вы правильно настроили ваш проект с использованием модулей Go. Вот шаги, которые помогут вам сделать это:

1. Создайте файл go.mod:
   Если у вас еще нет файла go.mod, выполните команду в терминале в директории вашего проекта:

   
   go mod init example.com/yourmodule
   
   Замените example.com/yourmodule на подходящее имя для вашего модуля.

2. Добавьте зависимость:
   В вашем коде используется пакет golang.org/x/net/html. Чтобы добавить эту зависимость, выполните команду:

   
   go get golang.org/x/net/html
   
   Это добавит нужный пакет в ваш файл go.mod.

3. Проверьте файл go.mod:
   Убедитесь, что ваш файл go.mod выглядит примерно так:

   
   module example.com/yourmodule

   go 1.17

   require golang.org/x/net v0.7.0 // замените на последнюю версию
   
4. Создайте HTML-файл:
   Убедитесь, что у вас есть файл index.html в той же директории, что и ваш main.go. Этот файл должен содержать CSS и JS ссылки, которые вы хотите скачать.

5. Запустите вашу программу:
   Теперь вы можете запустить вашу программу с помощью команды:

6. Скомпилируйте программу: Используйте команду go build, чтобы скомпилировать ваш файл. Это создаст исполняемый файл с именем, соответствующим имени директории (или вы можете указать имя файла). Например:

   go build -o myprogram main.go

   
   go run main.go
   
Если все сделано правильно, программа должна извлечь ссылки из index.html и скачать соответствующие CSS и JS файлы в текущую директорию.

Если возникнут ошибки, пожалуйста, предоставьте текст ошибки, и я помогу вам их устранить!
